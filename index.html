About EzGPG

EzGPG is a beginer friendly aproch to PGP and the GPG tool on MacOS. EzGPG requires gnupg to be pre-installed on your system.
If you dont hvae gnupg installed you can download it from HERE using Homebrew.


Whats PGP and GPG?

PGP stands for Pretty Good Privacy and is a data encryption and decryption program that provides people often use to prove their identity online. GPG or gnupg is a free software that is used to create PGP keys. However GPG is a command line tool and can be difficult to use for beginners. This is why I created EzGPG which simplifies the commands into numbered options.


Where can use EzGPG

You can thereticly use EzPGP anywhere you can install gnupg. However I have only tested it on MacOS. We are a free, open source software and you can find our code on github if you would like to build EzPGP for another OS.



How was it made?

EzGPG is coded entirely in Python3 however PGP, gnupg and GPG are made using other languages. EzGPG used the commands built into gnupg further simplify PGP encryption and decryption.
